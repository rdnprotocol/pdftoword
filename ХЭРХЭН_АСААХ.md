# Програмыг хэрхэн асаах вэ

## Хамгийн энгийн арга (Зөвлөмжтэй)

### Windows Explorer дээр:
**`START.bat`** файлыг хоёр удаа дарах (double-click)

### PowerShell дээр:
```powershell
.\start.ps1
```

Эсвэл:
```powershell
.\START.bat
```

Энэ нь:
- Frontend суулгаагүй бол автоматаар суулгана
- Backend, Frontend, Electron-ийг нэг дор эхлүүлнэ
- Гурван цонх нээгдэнэ

## Бусад аргууд

### Арга 1: Бүгдийг нэг дор

```bash
БҮГДИЙГ_ЭХЛҮҮЛЭХ.bat
```

### Арга 2: Тус тусад нь (3 терминал)

**Терминал 1 - Backend:**
```bash
run_backend.bat
```

**Терминал 2 - Frontend:**
```bash
run_frontend.bat
```

**Терминал 3 - Electron:**
```bash
run_electron.bat
```

## Шалгах

Програм зөв ажиллаж байгаа эсэхийг шалгах:

1. **Backend** цонхонд: `Uvicorn running on http://127.0.0.1:8000` гэж харагдах ёстой
2. **Frontend** цонхонд: `Compiled successfully!` гэж харагдах ёстой
3. **Electron** цонх: Desktop апп нээгдэнэ

## Асуудал гарвал

### Backend алдаа гарвал:
- Python суулгагдсан эсэхийг шалгах
- `cd backend && python main.py` гэж гараар ажиллуулах

### Frontend алдаа гарвал:
- `cd frontend && npm install` гэж дахин суулгах
- Node.js суулгагдсан эсэхийг шалгах

### Electron нээгдэхгүй бол:
- Frontend эхлээд дууссаныг хүлээх (Compiled successfully!)
- Дараа нь Electron ажиллуулах

## Програмыг хаах

Бүх цонхыг хаах:
- Backend цонх: `Ctrl+C` дараад цонхыг хаах
- Frontend цонх: `Ctrl+C` дараад цонхыг хаах  
- Electron цонх: Зүгээр л хаах

Эсвэл бүх cmd цонхыг хаах.
