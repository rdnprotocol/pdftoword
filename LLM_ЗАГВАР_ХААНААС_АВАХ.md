# LLM Загвар Хаанаас Авах Вэ?

> **Хуулах заавар:** [LLM_ЗАГВАР_ХУУЛАХ_ЗААВАР.md](LLM_ЗАГВАР_ХУУЛАХ_ЗААВАР.md) файлыг уншина уу.

## Монгол хэл дэмжих LLM загварууд

Монгол хэл дэмжих LLM загварууд хязгаартай байдаг. Дараах сонголтууд байна:

## Сонголт 1: Hugging Face (Зөвлөмжтэй)

Hugging Face нь олон LLM загваруудыг агуулдаг платформ юм.

### Алхмууд:

1. **Hugging Face руу орох:**
   - https://huggingface.co/

2. **Хайх:**
   - "mongolian" эсвэл "mongolia" гэж хайх
   - Эсвэл олон улсын загваруудыг хайх (Llama, Mistral, Qwen гэх мэт)

3. **.gguf форматтай загваруудыг олох:**
   - Загварын хуудас дээр "Files" таб дарна
   - `.gguf` өргөтгөлтэй файлуудыг хайх
   - Жишээ: `model.gguf`, `q4_0.gguf`, `q8_0.gguf` гэх мэт

4. **Татаж авах:**
   - `.gguf` файлыг татаж авах
   - `backend/models/` хавтас руу хуулах
   - Нэрийг `mongolian.gguf` болгох

### Жишээ загварууд:

- **Llama 2/3** - Зарим хувилбарууд монгол хэл дэмждэг
- **Mistral** - Олон улсын загвар
- **Qwen** - Азийн хэлүүдийг сайн дэмждэг

## Сонголт 2: TheBloke (Quantized Models)

TheBloke нь олон LLM загваруудыг quantized (.gguf) форматтайгаар танилцуулдаг.

### Алхмууд:

1. **TheBloke-ийн Hugging Face профил:**
   - https://huggingface.co/TheBloke

2. **Хайх:**
   - "mongolian" эсвэл олон улсын загваруудыг хайх
   - Quantized хувилбаруудыг сонгох

3. **Татаж авах:**
   - `Q4_K_M.gguf` эсвэл `Q8_0.gguf` хувилбаруудыг сонгох (сайн баланс)
   - Татаж аваад `backend/models/` руу хуулах

## Сонголт 3: Олон улсын загварууд (Монгол хэл дэмжих)

Олон улсын загварууд заримдаа монгол хэлийг дэмждэг:

### Mistral:

- Mistral-7B загварууд

### Qwen:

## Татаж авах заавар

### Hugging Face-аас татаж авах:

**Арга 1: Hugging Face Hub (Python)**
```bash
pip install huggingface-hub
```

```python
from huggingface_hub import hf_hub_download

# Загварыг татаж авах
model_path = hf_hub_download(
    repo_id="TheBloke/Llama-2-7B-Chat-GGUF",
    filename="llama-2-7b-chat.Q4_K_M.gguf",
    local_dir="backend/models"
)
```

**Арга 2: Шууд татаж авах**
1. Hugging Face дээр загварын хуудас руу орох
2. "Files" таб дарна
3. `.gguf` файлыг татаж авах
4. `backend/models/` хавтас руу хуулах

## Хэмжээ сонгох

LLM загварууд их хэмжээтэй байдаг:

- **7B параметр:** ~4-8 GB (зөвлөмжтэй)
- **13B параметр:** ~8-16 GB
- **70B параметр:** ~40+ GB (их хэмжээтэй)

**Quantization хувилбарууд:**
- `Q4_K_M`: Бага хэмжээ, сайн чанар (зөвлөмжтэй)
- `Q8_0`: Илүү том, илүү сайн чанар
- `F16`: Бүр хэмжээ, хамгийн сайн чанар

## Одоогийн байдал

**Хэрэв LLM загвар олохгүй бол:**
- Програм ажиллахгүй гэсэн үг биш
- Энгийн текст цэвэрлэлт хийх болно
- OCR ажиллана

## Тэмдэглэл

- LLM загварууд их хэмжээтэй байдаг (GB-аар)
- Татаж авахад удаан хугацаа шаардаж болно
- Интернет холболт шаардлагатай
- Диск зай хангалттай эсэхийг шалгах
