# Office AI Helper - Эхлэх Заавар

## Шаардлагатай зүйлс

1. **Python 3.8+** суулгасан байх
2. **Node.js 16+** суулгасан байх
3. **Tesseract OCR** монгол хэлний багцтайгаар суулгасан байх
4. **Poppler** PDF хөрвүүлэхэд зориулсан
5. (Сонголттой) **llama.cpp** болон монгол LLM загвар

## Суулгалт (Нэг удаа)

### 1. Backend суулгалт

**Windows дээр:**
```bash
setup_backend.bat
```

**Эсвэл гараар:**
```bash
cd backend
pip install -r requirements.txt
```

### 2. Frontend суулгалт

**Windows дээр:**
```bash
setup_frontend.bat
```

**Эсвэл гараар:**
```bash
cd frontend
npm install
```

### 3. Tesseract тохируулга (PATH-д байхгүй бол)

`backend/config.py` файлыг нээж, `TESSERACT_CMD` замыг тохируулна уу.

Жишээ:
```python
TESSERACT_CMD = r"C:\Program Files\Tesseract-OCR\tesseract.exe"
```

### 4. (Сонголттой) LLM тохируулга

- llama.cpp татаж авах: https://github.com/ggerganov/llama.cpp
- `main.exe` файлыг `backend/llama.cpp/` хавтасанд байрлуулах
- Монгол LLM загвар (.gguf) файлыг `backend/models/` хавтасанд байрлуулах

## Програмыг ажиллуулах

### Арга 1: Хөгжүүлэлтийн горим (Зөвлөмжтэй)

**Терминал 1 - Backend:**
```bash
run_backend.bat
```

Эсвэл:
```bash
cd backend
python main.py
```

Backend нь http://127.0.0.1:8000 дээр ажиллана.

**Терминал 2 - Frontend:**
```bash
run_frontend.bat
```

Эсвэл:
```bash
cd frontend
npm start
```

React апп нь http://localhost:3000 дээр ажиллана.

**Терминал 3 - Electron:**
```bash
run_electron.bat
```

Эсвэл:
```bash
cd frontend
npm run electron-dev
```

### Арга 2: Бүтээгдэхүүнийг бэлтгэх

1. React аппыг бэлтгэх:
```bash
cd frontend
npm run build
```

2. Electron аппыг багцлах:
```bash
npm run electron-pack
```

Гаралт: `frontend/dist/Office AI Helper Setup.exe`

## Хэрэглээ

1. Програмыг нээх
2. "Файл сонгох" товчийг дараад PDF эсвэл зураг (PNG/JPG) сонгох
3. "OCR + Засах" товчийг дараад боловсруулах
4. Зассан бичвэрийг харах
5. "Word-д татах" товчийг дараад Word баримт бичиг болгон экспортлох

## Асуудал шийдвэрлэх

### Tesseract олдсонгүй

- Tesseract суулгах: https://github.com/UB-Mannheim/tesseract/wiki
- Монгол хэлний багц суулгах шаардлагатай
- PATH-д нэмэх эсвэл `backend/config.py` дээр замыг тохируулах

### Poppler олдсонгүй

- Татаж авах: https://github.com/oschwartz10612/poppler-windows/releases
- `bin/` хавтасыг PATH-д нэмэх

### Backend холболтын алдаа

- Backend нь http://127.0.0.1:8000 дээр ажиллаж байгаа эсэхийг шалгах
- Галт хананы тохиргоог шалгах

### LLM ажиллахгүй байна

- LLM байхгүй тохиолдолд програм нь энгийн бичвэр цэвэрлэлт рүү шилжинэ
- `backend/llama.cpp/` хавтаст llama.cpp executable байгаа эсэхийг шалгах
- `backend/models/` хавтаст загварын файл байгаа эсэхийг шалгах
