# LLM (llama.cpp) Тохируулга - Сонголттой

## Энэ юу вэ?

LLM (Large Language Model) нь OCR-ийн алдааг засах, албан ёсны оффис хэлбэрт шилжүүлэхэд ашиглана.

## Хэрэв LLM байхгүй бол?

**Асуудалгүй!** Програм автоматаар энгийн текст цэвэрлэлт рүү шилжинэ:
- Хэт их зайг арилгах
- Мөрүүдийг цэгцлэх
- Үндсэн форматлах

LLM байхгүй тохиолдолд програм ажиллахгүй гэсэн үг биш!

## LLM суулгах (Хэрэв хүсвэл)

### Алхам 1: llama.cpp татаж авах

1. https://github.com/ggerganov/llama.cpp/releases руу орох
2. Windows хувилбарыг татаж авах (жишээ: `llama-bXXXX-w64-avx2.zip`)
3. Задрах
4. `main.exe` файлыг олох
5. `backend/llama.cpp/` хавтас үүсгэх (хэрэв байхгүй бол)
6. `main.exe`-ийг `backend/llama.cpp/` хавтас руу хуулах

### Алхам 2: Монгол LLM загвар татаж авах

1. Монгол хэл дэмжих LLM загвар олох (.gguf формат)
2. Загварыг `backend/models/` хавтас руу хуулах
3. Жишээ нэр: `mongolian.gguf` эсвэл `mon-llm.gguf`

### Алхам 3: Тохируулга

`backend/config.py` файлд замыг шалгах:

```python
MODEL_PATH = Path(__file__).parent / "models" / "mongolian.gguf"
LLAMA_CPP_PATH = Path(__file__).parent / "llama.cpp" / "main.exe"
```

Хэрэв таны загварын нэр өөр бол MODEL_PATH-ийг өөрчлөх:

```python
MODEL_PATH = Path(__file__).parent / "models" / "таны_загварын_нэр.gguf"
```

## Шалгах

LLM зөв тохируулагдсан эсэхийг шалгах:

```bash
cd backend
python -c "from ai_fix import LLMCorrector; c = LLMCorrector(); print('Model:', c.model_path); print('Exists:', c.model_path.exists() if c.model_path else False)"
```

## Тэмдэглэл

- LLM загвар их хэмжээтэй байж болно (GB-аар)
- Эхний удаа ашиглахад удаан хугацаа шаардаж болно
- Хэрэв LLM байхгүй бол програм ажиллахгүй гэсэн үг биш - энгийн цэвэрлэлт хийх болно

## Монгол LLM загвар хаанаас олох вэ?

Одоогоор монгол хэл дэмжих LLM загварууд хязгаартай байна. Та дараах сонголтуудыг туршиж болно:

1. **Олон улсын загварууд** (монгол хэл дэмжих):
   - Llama 2/3 загварууд (зарим нь монгол хэл дэмждэг)
   - Mistral загварууд

2. **Өөрийн загвар бэлтгэх** (дэвшилтэт):
   - Монгол бичвэр дээр fine-tuning хийх

3. **Одоогийн байдлаар LLM байхгүй бол**:
   - Програм энгийн текст цэвэрлэлт хийх болно
   - Энэ нь ихэнх тохиолдолд хангалттай байдаг

## Дүгнэлт

**LLM нь сонголттой!** Хэрэв танд LLM байхгүй бол програм ажиллахгүй гэсэн үг биш. Энгийн текст цэвэрлэлт хийх болно.

Хэрэв та LLM суулгахыг хүсвэл дээрх алхмуудыг дагана уу.
